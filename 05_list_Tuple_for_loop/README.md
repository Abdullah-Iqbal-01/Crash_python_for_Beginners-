## Lists

### What is a List?

A **list** is a collection of ordered, changeable, and indexed items. Lists allow you to store multiple values in a single variable. Lists are defined using square brackets `[]`.

#### Example:

```python
my_list = [1, 2, 3, 4, 5]
print(my_list)
